## SPAフレームワーク無しでwebpackを使用するサンプル

### 以下の要件を元に作成

- buildするとbundle, minifyされたコードが生成される
- HumanReadableなコードでデバッグできる
- typescriptが使用出来る
- jsから読み込む画像ファイルがbuildによって自動で出力される
- jsから読み込むscssファイルがbuildによって自動で出力される


### ビルド

```
npm run build
```
または
```
yarn build
```

### デバッグ

```
npm run start
```
または
```
yarn start
```
